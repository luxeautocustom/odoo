Odoo 15 Section Sub Total (Enterprise)

This module allows user to show subtotal of section on order lines.

Note: This module is tested and working in Odoo Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • We can add section as usual.
  • We can see total of each section of sale order line.
  • We can see total of each section of purchase order line.
  • we can see separate table with section name and total as well.

Available for Community

How to use

  • sale order line with section subtotal.


  • section table with name and total of each section.


  • purchase order line with section subtotal.


  • section table with name and total of each section.
