This module allow to define sections in products and move sale order line to the define section in product.

Features

  • Select section in product form.
  • Now when you create the sale order and save it it will move all sale order line to the respected section define in products.
  • It will create section on sale order line dynamically. no need to create section manually.
  • Once you create invoice from sale order invoice will also follow the same sale order structure.
  • Select section in product form.


  • You can see the product group by with the two sections in below screenshot


  • Create the sale order and select the products.


  • Now once you save the sale order you can see the section created dynamically and sale order lines moves to the respected section.


  • Invoice follows the same sections structure.
