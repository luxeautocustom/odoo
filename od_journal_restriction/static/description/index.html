Journal Restriction For Users

Set allowed users inside the journal and this journals will be accessible for allowed users. If no users is set, it will be accessible for all users.

Features

  • You can restrict specific journals to specific users.
  • You can configure a user in a journal, so users can access that journal.
  • Can restrict all the journals like invoice, bill, cash, bank, sale & purchase.
  • If no user is configured then will be accessible for all users.
  • Either configure journal in users or users inside the journal.
  • Overview
  • Screenshots

Overview

By default in odoo, the journals will be accessible for all the users. There might be some use cases where we have to limit this visibility, so that particular users can use some journals in the company. This module will help users to control this by setting allowed users, so that the journal can be accessed and operated by the allowed users.
